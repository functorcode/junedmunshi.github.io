City Guide

City guide mobile app help you to effectively plan trip. You just need to select attractions you want to visit that day and app will automatically create itinerary of the day. You can also discover popular attractions, food and nightlife options.

Featuers

  • Automatically create optimal route for multiple location
  • Works with public transit option
  • Curated list of places to visit with

Technology

  • Android
  • Parse.io for API backend
  • Google Map api integration
  • In house algorithm developed to calculate optimal path
  • Angular JS, Bootstrap, HTML for admin panel
