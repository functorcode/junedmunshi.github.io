How to install github project as module into nodejs project

Posted by Juned Munshi on August 16, 2017
  1. Add following under “dependencies” into package.json 
     "packagename":"username/reponame"

    For example,

     "dependencies": {
 "fb-bot-framework":"junedmunshi/fb-bot-framework"
  }

  2. Install 
    npm install
